Virtual Library

This project is an exercise in JavaScript object oriented programming and browser memory storage.

Each book entry is created using a JavaScript object constructor and has 5 properties: "Title" (string), "Author" (string), "Pages" (number), "URL" (string), and "read" (boolean). Each book object shares methods defined in its object prototype to access and change its properties.

Click the "Add +" button to add a new book to the libray. Click the "Read" button on each book to change its read status. Click the "Delete" button on each book to remove it from the collection.

This web app uses your browser's "localStorage API" to store your books on your device's memory. You can test this feature by making changes and refreshing the page or closing and reloading it. To see what data is currently stored in your browser's local memory for this webpage, press F12 to open developer tools, find the "Application" tab (may be hidden in an arrow), expand the "Local Storage" menu on the left panel, and select this wepage's URL. Your books will remain in memory until you perform "clear browsing data" or until you click the provided "Clean Reset" button.

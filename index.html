Virtual Library

This project is an exercise in JavaScript object oriented programming and cloud data storage. The focus for this project is on back end development, so the visual presentation is left intentionally sparse.

Please select from the following options:


Cloud Storage:

uses Google's Firebase cloud database service to store your books in a remote server (AKA "The Cloud")

Local Storage:

uses your browser's localStorage API to store your books in your device's memory.

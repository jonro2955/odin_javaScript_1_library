Virtual Library

An exercise in JavaScript object oriented programming.
Each book entry is created using a JavaScript Object Constructor, and contains
5 properties: "title" (a string), "author" (a string), "pages" (a number), "URL" (a string), and "read" (a boolean).
Each book object shares methods defined in its "Book" object prototype to access and change its properties.
Click the "Add +" button to add a new book to the libray.
Click the "Read" button on each book to change its read status.
Click the "Delete" button on each book to remove it from the collection.

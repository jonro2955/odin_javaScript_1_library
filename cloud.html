Cloud Storage Library

In this version, data is stored retrieved from the cloud using Google's Firebase. Firebase is a free online platform for creating mobile and web applications, and includes a real time database to save user data. Changes made on this page will update the database in real time and will persist throughout page reloads. Each book is stored as an object with 5 key-value pair properties: Title (string), Author (string), Pages (number), Link (string), and Read (boolean). Try adding a new book by clicking the Add New Book + button!

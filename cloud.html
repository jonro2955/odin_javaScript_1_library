Cloud Storage Library

This webpage is an exercise in JavaScript object oriented programming and cloud data storage. The focus for this project is on back end development, so the visual presentation is left intentionally sparse.

Each book is a JavaScript object with 5 properties: "Title" (string), "Author" (string), "Pages" (number), "URL" (string), and "Read" (boolean). Object methods are defined in its prototype to be shared amongst all instances.

This page uses Google's Firebase cloud database service to store this webpage's data in the cloud.

Click the "Add +" button to add a new book to the library. Click the "Read" button on each book to change its read status. Click the "Delete" button on each book to remove it from the collection.

Cloud Storage Library

Click the "Add +" button to add a new book to the library. Click the "Read" button on each book to change its read status. Click the "Delete" button on each book to remove it from the collection.

In this version, your books will be stored in a remote Google server as provided by Google's Firebase service (free version). You can test this feature by making changes and then reloading or closing and reopening this page. Any changes you make here will change the cloud database in real time. The button labeled "Reset With 3 Sample Books" will restore this site back to its initial state by running a script that will reset the cloud database with only the 3 selected sample books.
